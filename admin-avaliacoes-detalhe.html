📝 Resumo do Trabalho
Abstract académico

Este trabalho apresenta uma abordagem inovadora para o desenvolvimento de sistemas de gestão hospitalar em ambientes distribuídos, com foco na interoperabilidade entre unidades sanitárias e na consistência de dados clínicos em tempo real. A arquitectura proposta baseia-se em microserviços comunicantes via protocolo gRPC, com persistência híbrida combinando bases de dados relacionais e não-relacionais para optimização de desempenho.

O sistema foi desenvolvido e testado no contexto do sistema de saúde angolano, considerando as limitações de conectividade e a diversidade de infraestrutura tecnológica existente. Os resultados demonstram uma redução de 40% no tempo de acesso a registos clínicos e uma disponibilidade de 99.2% durante os testes realizados.
